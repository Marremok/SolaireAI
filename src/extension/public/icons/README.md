# Extension Icons

The extension requires three icon sizes:
- **icon16.png** - 16x16 pixels (toolbar icon)
- **icon48.png** - 48x48 pixels (extension management page)
- **icon128.png** - 128x128 pixels (Chrome Web Store)

## How to Create Icons

### Option 1: Use an online tool
1. Go to https://www.figma.com or https://www.canva.com
2. Create a 128x128px canvas with a dark background (#000000)
3. Add a sun icon (☀️) or the SolaireAI logo in primary blue (#1C9CF0)
4. Export as PNG at 128x128, 48x48, and 16x16

### Option 2: Use an existing logo
If you have the SolaireAI logo:
1. Resize to 128x128, 48x48, and 16x16 pixels
2. Save as PNG files with the names above
3. Place them in this directory

### Temporary Solution
For development, you can use placeholder icons:
- Create simple colored squares (blue background)
- Add "SA" text in white
- This is enough for local testing

The icons should match the SolaireAI brand:
- Primary color: #1C9CF0 (blue)
- Background: #000000 (black) or transparent
- Style: Modern, clean, minimalist
